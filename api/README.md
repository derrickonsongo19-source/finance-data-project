# Personal Finance Analytics API

## Overview
FastAPI-based REST API for the Personal Finance Analytics Platform. This API provides real-time financial insights, ML-powered predictions, and budget recommendations.

## Features
- Real-time financial summary
- Transaction management
- Spending predictions using ML
- Budget recommendations
- Historical data analysis
- MLflow experiment tracking

## Prerequisites
- Python 3.12+
- DuckDB database with finance data (see main project)
- Virtual environment (recommended)
